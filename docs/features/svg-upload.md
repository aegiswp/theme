# SVG Upload

Secure SVG media upload settings are managed by the **Aegis companion plugin** at **Aegis → Settings**.

## Documentation

See the full SVG upload guide:

**[Aegis Plugin — SVG Upload](../../plugins/aegis/docs/features/svg-upload.md)**

SVG gating when upload is disabled requires **Aegis Pro** — see [SVG Gating](../../plugins/aegis-pro/docs/features/svg-gating.md).

The **SVG block variation** toggle is separate, under **Aegis → Blocks**.
