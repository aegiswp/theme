## Applying a Variation

1. Open the Site Editor (**Appearance → Editor**).
2. Click the **Styles** icon (half-filled circle).
3. Click **Browse styles**.
4. Select a variation from the grid.
5. Click **Save** to apply.

## How Variations Affect Design Tokens

Each variation may modify:

| Token Category | Example Changes |
|----------------|----------------|
| Primary colors | Different hue for the primary palette |
| Background | Page and section background colors |
| Text colors | Heading, body, and link colors |
| Accent colors | Button backgrounds, link highlights |
| Font pairing | Some variations adjust font families or weights |
| Border colors | Card and divider border tones |

## Creating Custom Variations

Developers can add custom style variations:

1. Create a JSON file in the `styles/` directory.
2. Follow the `theme.json` schema for settings and styles overrides.
3. Include a `title` property for the display name.
4. The variation appears automatically in the style browser.

Example structure:

```json
{
    "$schema": "https://schemas.wp.org/trunk/theme.json",
    "version": 3,
    "title": "My Custom Variation",
    "settings": {
        "color": {
            "palette": []
        }
    },
    "styles": {
        "color": {
            "background": "#ffffff",
            "text": "#1a1a1a"
        }
    }
}
```
